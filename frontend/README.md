# Frontend

## Dev

1) Install

   npm install

2) Optional API base

   Create a .env file with:

   VITE_API_URL=http://localhost:5000

   If not set, defaults to http://localhost:5000

3) Run

   npm run dev

## Build

- npm run build
- npm run preview
